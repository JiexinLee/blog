###### 开发前的配置要求

1. Flutter 2.15 <= 3.0
2. Dart 

###### **Installation**

1. Clone the repo

```sh
git clone https://github.com/JiexinLee/blog.git
```
2. Run flutter pub get in your terminal under the blog directory

```sh
flutter pub get
```
